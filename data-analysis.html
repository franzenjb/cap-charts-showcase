Best Practices Summary

📊 For Quarterly Comparisons:

  • Use Column/Bar Charts when you want to compare individual quarters
  • Use Line Charts when you want to show trends over time
  • Use Area Charts when you want to show accumulation

📈 For Growth Metrics:

  • Use Waterfall Charts to show how you built to a total
  • Use Sparklines for compact trend display
  • Use Growth Rate Charts to show percentage changes

🎯 For Performance Metrics:

  • Use Gauges/Speedometers for single KPIs against targets
  • Use Bullet Charts to compare actual vs target
  • Use Inverse Charts for "lower is better" metrics

🌍 For Impact Metrics:

  • Use Maps for geographic distribution
  • Use Icon Arrays to make numbers feel human
  • Use Donut/Pie Charts for proportional breakdowns
